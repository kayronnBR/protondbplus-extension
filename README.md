# protondbplus-extension
Extension to redirect your search to ProtonDB Plus, made by Gemini.

How to Install the Extension Manually
Follow these steps to install the plugin using Developer Mode:

1. Prepare the Files
Download the extension file (usually in .zip format).

Extract the contents of the .zip file into a specific folder on your computer.

Note: Do not delete or move this folder after installation, as the browser needs it to run the extension.

2. Enable Developer Mode
Open your browser and right-click the Extensions icon (the puzzle piece) or go to the menu and select Manage Extensions.

In the top-right corner, toggle the Developer mode switch to ON.

Click the Load unpacked button that appears in the top menu.

Select the folder where you extracted the extension files.

3. Updates and Maintenance
Confirm Installation: Once loaded, the extension will appear in your list. Click Update to ensure everything is synced.

Checking for Updates: You can check the current version at the top of the extension site (e.g., Extension 0.1).

How to Update: If a new version is released, download the new .zip and extract it, overwriting the files in the existing folder. Then, go back to the "Manage Extensions" page and click Update.
